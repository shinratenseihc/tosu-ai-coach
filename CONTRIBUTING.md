# Contribuer

1. Ouvre une issue pour les changements importants.
2. Crée une branche courte et ciblée.
3. Ajoute ou adapte les tests.
4. Exécute `npm test` et `npm run check`.
5. Décris le comportement avant/après dans la pull request.

Les données personnelles, historiques, logs et secrets sont interdits dans les commits. En contribuant, tu acceptes la licence MIT. Le coach peut chambrer ; les discussions entre contributeurs restent respectueuses.
