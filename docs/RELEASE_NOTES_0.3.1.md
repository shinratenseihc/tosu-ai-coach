# TOSU AI Coach 0.3.1

Cette version durcit la sécurité du service local : seul l’état nécessaire à l’overlay reste accessible cross-origin, tandis que les actions refusent les origines web externes.

Elle fiabilise aussi les fondations du coach avec une validation de configuration isolée, un gestionnaire d’état testé, un client osu! lié aux identifiants et une vérification syntaxique automatique de tous les fichiers JavaScript. Les options en ligne qui ne sont pas encore actives sont désormais clairement indiquées comme « bientôt » dans le dashboard.
